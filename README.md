# AI-POWERED CCTV THAT DETECTS FIGHTS, ACCIDENTS AND WEAPONS

## Project Overview
This project presents an **AI-powered smart CCTV surveillance system** capable of automatically detecting **violent fights, road accidents, and weapons** in real-time video streams.  
The system leverages **deep learning and computer vision techniques** to enhance public safety by providing early alerts and actionable insights to authorities.

Traditional CCTV systems require continuous human monitoring, which is inefficient and error-prone. Our proposed solution automates surveillance by intelligently analyzing video feeds and identifying suspicious or dangerous activities.

---

## Problem Statement
Manual monitoring of CCTV footage is time-consuming and unreliable, especially in crowded or high-risk areas.  
Delayed detection of fights, accidents, or weapon usage can lead to severe consequences.

This project aims to:
- Reduce human dependency in surveillance
- Detect violent and dangerous activities in real time
- Improve response time during emergencies

---

## Objectives
- To detect **violent activities (fights)** using deep learning models
- To identify **weapons** in CCTV footage using object detection
- To detect **road accidents** and trigger alerts
- To provide a scalable and automated surveillance solution

---

## Modules

### 1. Violence & Weapon Detection Module
- Detects physical fights and aggressive behavior
- Identifies weapons such as knives and guns
- Uses YOLO-based object detection and deep learning classifiers

### 2. Accident Detection & Notification Module
- Detects road accidents from CCTV footage
- Helps in faster emergency response
- Can be extended to send alerts to authorities

### 3. Future Enhancements
- Integration with real-time alert systems (SMS / Email)
- Face recognition for offender identification
- Cloud-based monitoring dashboard
